this program helps you save details about products in a shop.you can enter up to 10 products. for each, you type its id, name, how many are in stock, and what price it has.after entering, it shows all products in a list on the screen.the program adds up all the quantity to show total stock.it multiplies quantity and price for each product to show the total value of all products.it finds which product has the highest value (most worth) in stock.you can search for a product by typing its id, and the program will show its details if it exists.only arrays and loops are used. no functions or extra things.everything is easy to see and use.
